### Local Setup Instructions

1. **Navigate to the project directory**
   - Change into the directory where you cloned this repository:

```bash
cd Transcript Tool
```

   (Or navigate to wherever you cloned the repository on your system)

2. **Create and/or activate the virtual environment**
   - If the `venv-webapp` virtual environment already exists, activate it:

```bash
source venv-webapp/bin/activate
```

3. **Run the web application**

```bash
python3 app.py
```

4. **Deactivate the virtual environment (optional)**

```bash
deactivate
```
